# Generates packets to parse for 2018 Hack the North

The requirements are at http://pad/test-packet-comms-pad

## Usage
`./bin/generate > ./.packets`

## Configuration
`vi ./config/packet_comms.yml`

## Hack the North Challenge 2018
### The Challenge
Here's the wiki of the mentioned encoding:
https://en.wikipedia.org/wiki/Financial_Information_eXchange

You will be provided with a string input (multiple packets) that need to be decoded and processed.
The goal is to decode the message of each packet and process the operation between accumulator and next message.
There are multiple accumulators that need to be processed.
An accumulator is initialized to 0.

### The Data
The format of a packet's message is as follows:
- `message: (key=value)|*`
- `key: Number`
- `=: "="`
- `value: String, Number`

Example packet: `8=HtN|3=12|2=ADD|5=A`.
The output should be the result of the operations for each accumulator.
A message may not have its keys in order, but always starts with the header
    
Expected keys:

- `8 = header - String`
- `2 = instruction - Enum(ADD, SUB, MUL)`
- `3 = data - int`
- `5 = accumulator - String`

### How to Play
Example input packets:
- `8=HtN|3=12|2=ADD|5=A|8=HtN|2=SUB|5=B|3=100|8=HtN|5=A|2=MUL|3=2|8=HtN|2=SUB|5=B|3=1`

Example output string:
- `A = 24, B = -101`

Trace:
- `8=HtN|3=12|2=ADD|5=A|`8=HtN|2=SUB|5=B|3=100|8=HtN|5=A|2=MUL|3=2|8=HtN|2=SUB|5=B|3=1
  - A = 0 + 12 = 12
- 8=HtN|3=12|2=ADD|5=A|`8=HtN|2=SUB|5=B|3=100|`8=HtN|5=A|2=MUL|3=2|8=HtN|2=SUB|5=B|3=1
  - B = 0 - 100 = -100
- 8=HtN|3=12|2=ADD|5=A|8=HtN|2=SUB|5=B|3=100|`8=HtN|5=A|2=MUL|3=2|`8=HtN|2=SUB|5=B|3=1
  - A = 12 * 2 = `24`
- 8=HtN|3=12|2=ADD|5=A|8=HtN|2=SUB|5=B|3=100|8=HtN|5=A|2=MUL|3=2|`8=HtN|2=SUB|5=B|3=1`
  - B = -100 - 1 = `-101`

### Winning the Game
The first to announce the results of the accumulators wins!
